Update 1

2022-04-23

업데이트는 생성(CREATE)과 컨텐츠 읽기(READ)가 합쳐져 난이도가 꽤 높은 작업이다. 그래도 이를 하고 나면 기본은 마칠 수 있으리라 😣 생각해보자

기존 데이터를 수정하고 업데이트해 보자.

  1. 초기값이 없는 변수를 하나 설정한다.
  2. Update 링크를 생성한 후 READ 모드일 때 적용될 수 있도록 작성.
    • home에 있는 경우 update 링크는 필요하지 않으므로 상세보기로 들어간 경우에만 링크가 나타나게 된다.
    • contextControl은 값이 없다가 mode가 READ로 세팅된 경우에만 값이 적용되기 때문이다.
  3. 클릭을 하면 일어나는 기본 값(reload)을 제거하고, 설정될 mode의 값을 지정한다.
  4. mode가 'UPDATE'시 일어날 if else문을 추가한다.
    • content 변수에 Update 컴포넌트를 넣었지만, 컴포넌트를 정의하지 않았기 때문에 에러화면이 뜨게 된다.
  5. Update 컴포넌트의 내용을 구현한다.
    • onUpdate 라는 prop 생성이 필요하다.
  6. onUpdate, title, body의 props 설정
  7. Update는 수정이기 때문에, form 안에 기존의 내용이 들어가야 한다.
    • if문의 title을 Update 컴포넌트 안의 prop인 title에 적용한다.
  8. props를 해당 value의 값에 넣는다.
  9. 그렇게 입력하고 나면 수정할 수 있게 기존 내용이 들어가게 된다!
    • 하지만 props는 사용자가 component로 전달한 명령이므로 바로 수정이 되지 않는다.....
