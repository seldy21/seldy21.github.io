속성 PROPS

2022-04-17

컴포넌트에 속성을 갖게 하려면 어떻게 해야 할까?

복잡하지만 일단 따라해보자..

  1. Header 태그에 속성명을 넣고 넣고자하는 콘텐츠를 표기한다.
  2. 해당 함수 선언에 props 값을 넣고 return 안 콘텐츠에 {props.속성명} 표기
    • 중괄호를 꼭 써야한다.
    • 이 방법과 똑같이 article에도 콘텐츠를 넣었다.

이번에는 배열의 값을 속성으로 지정해보자

  1. 콘텐츠를 표시하는 함수 내에서 상수를 선언한다.
    • 배열로 작성해야 하며, (당연한 얘기지만) 텍스트는 '' 안에 작성해야 한다.
  2. 태그 내에 속성을 작성한다.
  3. 해당 함수에 props 작성 후 상수 선언을 한다. 그 안에는 배열로 들어갈 태그들을 넣는다.
    • 배열의 값이기 때문에 대괄호를 쓴다. 선언된 상수는 콘텐츠가 들어갈 자리에 {상수명} 으로 작성한다.
    • 각 태그들은 콤마로 구별한다.
  4. 배열의 값을 for문으로 적용한다.
  5. 변수를 선언하고 선언했던 상수 lis에 push 한다.
  6. 배열로 작성할 태그를 삭제하고, push 내에 작성한다.
  7. 완성 ㅠㅠ
