fontello 아이콘 연결하기

2022-04-14

fontello 아이콘은 사용하기 매우 쉽고 편하다. 원하는 아이콘만 다운받아 코드로 사용하면 된다. 약간의 꾸밈으로 전체적인 분위기를 아기자기하게 만들 수 있기에 아이콘의 역할은 중요하다.

fontello 사용하기

  1. fontello 사이트로 접속 https://fontello.com/해서 원하는 아이콘을 클릭
  2. 오른쪽 상단 빨간색 버튼 Download webfont 클릭
  3. 다운로드 후 압축풀기
  4. 생성된 파일 내 css 폴더에서 fontello.css 문서를 연결하고자 하는 html 문서에 link로 연결
  5. 폴더 내에서 demo.html 파일을 열면 각 아이콘의 클래스명을 확인할 수 있다.
  6. 삽입하고자 하는 위치에 i 태그를 써서 사용한다.
