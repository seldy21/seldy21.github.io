      범객체인 데이터라면, (예를 들어 Object, Array 등) 데이터를 복제하여 새로운 데이터로 변환하여 그 복제본을 사용하는 방식을 이용한다. 배열일 경우는 대괄호[]안에 넣어 복제한다.

      복제본은 {...value}로 표기한다.

      const newTopics = [...topics]

      이렇게 변수 topics의 복제본이 만들어진다.

      setTopics.push(newTopics)

      topics의 복제본에 변한 값이 적용된다.

      const [value,setValue] = useState([1]);
      newValue = [...value];　　　　　　----ⓐ
      newValue.push(2);　　　　　　　　---ⓑ
      setValue(newValue);　　　　　　　---ⓒ
      ⓐ 기존의 데이터를 복제하고 ⓑ 복제한 데이터에 값을 추가하여 변경하고 ⓒ변경한 데이터의 값을 setValue에 넣는다.
