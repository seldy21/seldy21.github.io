javascript event

2022-04-20

이벤트 : 브라우저 상에서 상태를 바꾸는 특정 행위.

이벤트 핸들러 : 이벤트가 발생했을 때 그에 대한 반응. 이벤트에 'on'을 붙여 동작을 처리한다.

이벤트 종류

  1. 마우스이벤트
    click 마우스 클릭 시
    dbclick 마우스 더블클릭 시
    mousedown 클릭하고 있을 때
    mouseup 클릭이 떼어졌을 때
    mousemove 마우스 움직이고 있을 시
    mouseover 마우스가 요소 위에 있을 때. (하위 노드에 영향 있음)
    mouseenter 마우스가 요소 위에 있을 때. (하위 노드에 영향 없음)
    mouseout 마우스가 요소를 벗어났을 때. (하위 노드에 영향 있음)
    mouseleave 마우스가 요소 위에 있을 때. (하위 노드에 영향 없음)
    contextmenu 마우스 오른쪽 버튼을 눌렀을 때
  2. 키 이벤트
    keydown 키를 누를 시
    keyup 키를 떼었을 시
    keypress 키를 누른 상태일 시
  3. 폼 이벤트
    focus 요소에 포커스가 놓였을 시
    blur 요소에서 포커스가 벗어났을 시
    change 요소의 값이 변경되었을 시
    submit submit 버튼을 눌렀을 시
    reset reset 버튼을 눌렀을 시
    select input이나 textarea 요소 안의 텍스트를 드래그하여 선택했을 시
  4. 로드 및 기타 이벤트
    load 페이지의 로딩이 완료되었을 시
    abort 이미지의 로딩이 중단되었을 시
    unload 페이지가 다른 곳으로 이동되었을 시
    resize 요소의 사이즈가 변경되었을 시
    scroll 스크롤바를 움직였을 시
