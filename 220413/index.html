기술블로그 만들기

2022-04-13

기술 블로그를 시작했다. 하루하루 배우는 것들을 기록하기 위함인데, 지금 나의 목표는 꾸준히, '즐겁게!'

'기본'은 가장 단순하지만 실전으로 지키기는 어려운 법이다.

github로 블로그 만들기

1. github로 repository 생성

2. username.github.io로 주소 설정 (ADD a README file 체크)

3. repository 생성 후 초록색 버튼 Code를 눌러 주소 복사

4. commit할 폴더를 디바이스에 생성하거나 열고 마우스 오른쪽 버튼 누르고 git bash here 클릭

5. clone 생성 명령어 입력

git clone 복사한 주소

6. clone한 폴더로 이동 후 파일 생성 명령어

cd username.github.io
echo "Hello World" > index.html

7. push 명령어 입력

git add --all
git commit -m "추가할 메시지"

8. 주소창에 username.github.ico 입력

9. 꾸밈의 값은 html과 css로 적용하여 완성
